Applicazione web per la gestione e il monitoraggio del budget giornaliero. Il progetto utilizza HTML5 con responsive design e è versionato tramite Git con repository remoto.

Si tratta di un'applicazione standalone in formato HTML che permette di tracciare le spese quotidiane e gestire le finanze personali direttamente dal browser. Il codice è completo e funzionante, con implementazione di PWA (Progressive Web App) come indicato dai meta tag mobili presenti nel file principale.
